This is a mobile app prototype built with Proto.io mobile app prototyping tool, then exported and edited in JS and HTML.

Access the app by going to the Github page here: https://dtbyrne.github.io/climaca/index.html.
Simply open the index.html file to get started using the application!
Use your cursor as a finger to swipe the screen on a computer, or you can simply open the link on your preferred mobile device!

This app was built for the resolution of an iPhone 8 in mind (about 375 x 667).
